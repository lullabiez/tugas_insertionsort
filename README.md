# Insertion Sort App

Aplikasi Command Line Interface (CLI) sederhana untuk mengurutkan data angka menggunakan metode **Insertion Sort** dengan visualisasi proses di setiap iterasinya.

## Cara Kerja Insertion Sort
Algoritma ini bekerja seperti cara kita mengurutkan kartu di tangan. Elemen diambil satu per satu, kemudian dibandingkan, lalu disisipkan ke posisi yang tepat pada bagian yang telah terurut.
